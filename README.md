<h2>Project Type</h2>
    <p>Backend Web Application</p>

    <h2>Purpose</h2>
    <p>Final capstone project for Thinkful’s Backend Engineering curriculum.</p>

    <h2>Project Overview</h2>
    <p>
        <strong>We Love Movies</strong> is a backend web application designed to manage and serve movie data efficiently. 
        Built as the culminating project in Thinkful’s backend program, this project demonstrates my ability to design RESTful APIs, work with relational databases, 
        and implement server-side logic using modern JavaScript technologies.
    </p>

    <h2>Key Features</h2>
    <ul>
        <li>Robust backend for managing movie data</li>
        <li>Efficient handling of CRUD operations for movie records</li>
        <li>Scalable database design with PostgreSQL</li>
    </ul>

    <h2>Technologies Used</h2>
    <ul>
        <li><strong>Node.js</strong> – Server-side JavaScript runtime</li>
        <li><strong>Express.js</strong> – Framework for building RESTful APIs</li>
        <li><strong>PostgreSQL</strong> – Relational database for storing movie information</li>
        <li><strong>Knex.js</strong> – SQL query builder for interacting with the database</li>
    </ul>

    <h2>Getting Started</h2>
    <ol>
        <li>Clone the repository</li>
        <li>Install dependencies:
            <pre><code>npm install</code></pre>
        </li>
        <li>Start the server:
            <pre><code>npm start</code></pre>
        </li>
        <li>Access the API at <code>http://localhost:3000</code></li>
    </ol>

    <h2>Learning Highlights / Skills Demonstrated</h2>
    <ul>
        <li>RESTful API design and implementation</li>
        <li>Database schema design and SQL querying</li>
        <li>Server-side routing and middleware with Express.js</li>
        <li>Integration with a relational database using Knex.js</li>
    </ul>
