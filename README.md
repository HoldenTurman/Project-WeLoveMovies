# We Love Movies – Thinkful Capstone Project

## Project Overview
**We Love Movies** is a backend web application created as the final capstone project for Thinkful’s Backend Engineering curriculum.  
This project demonstrates my ability to design RESTful APIs, manage relational databases, and implement server-side logic using modern JavaScript technologies.

## Purpose
The application is designed to manage and serve movie data efficiently. It showcases skills in backend development, database management, and collaborative software engineering practices.

## Key Features
- Robust backend for managing movie data
- Efficient CRUD operations for movie records
- Scalable database design using PostgreSQL

## Technologies Used
The project leverages modern backend technologies, including:
- **Node.js** – Server-side JavaScript runtime environment
- **Express.js** – Web framework for building RESTful APIs
- **PostgreSQL** – Relational database for storing movie information
- **Knex.js** – SQL query builder for database interactions
